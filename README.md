# SD Vendetta Battle Royale

## Project Documentation

### Overview
SD Vendetta is a Battle Royale game where players compete against each other until only one player remains. The game features fast-paced action, a variety of weapons, and strategic gameplay.

### Getting Started

#### Prerequisites
Make sure you have the following software installed:
- Unity 2020 or later
- Visual Studio 2019 or later
- Git

#### Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/23121998sd/SD-Vendetta-Game.git
   ```
2. Open the project in Unity.
3. Set up your environment by following the instructions in the setup file located in the `docs` folder.
4. Build and run the game to get started.

### Gameplay Instructions
- Use [WASD] to move around.
- Use the mouse to aim and shoot.
- Collect weapons and power-ups to enhance your abilities.
